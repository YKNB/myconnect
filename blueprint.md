# Address Book

## Overview

This application is a simple address book that allows users to manage a list of contacts. It is built with Angular and uses Firebase for the database and hosting.

## Style and Design

The application uses a simple and clean design. The layout is responsive and adapts to different screen sizes.

- **Font**: Sans-serif
- **Colors**: A simple color palette with blue for buttons and links.
- **Layout**: A table-based layout for the contact list and a form-based layout for adding and editing contacts.

## Features

- Add, edit, and delete contacts.
- View a list of contacts.
- Data persistence using Firebase Firestore.
- User authentication with email/password and Google Sign-In.
